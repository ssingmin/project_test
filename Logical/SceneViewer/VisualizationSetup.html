Conveyor Visualization with B&R Scene Viewer

  1. Start your conveyor controller (or simulation).
  2. Start B&R Scene Viewer.
  3. In the "File" menu, select "Open". Then, select "Visualization.scn".

  4. In the "PVI" menu, select "Options". Configure as follows:

    IP address (your controller's IP address)
    Task Name ST_Visu
    Variable name Data
    Size 3072
    Extra (empty)
    Additional connection parameters (empty)
    Timer mode enabled
    Refresh rate 25ms
    Timing variable (not applicable)

    To connect to a local ARsim instance, specify the IP address "127.0.0.1".

  5. In the "PVI" menu, select "Start PVI". The status bar should change from "OFFLINE" to "RUN", and the visualization should display the state of the system.

    If the connection cannot be established, or the visualization does not function correctly, verify the following:

    • The correct IP address was specified in PVI Options.
    • Networking is configured and cabled correctly.
    • Your controller is functioning correctly.
    • Your controller supports visualization.

  6. If desired, adjust the view. The controls are described in the Scene Viewer help file. The most frequently used controls are as follows:

    Toggle Full Screen
    Double-click the left mouse button.
    Zoom
    Rotate the mouse wheel.
    Pan
    Depress and hold the mouse wheel, and drag.
    Rotate
    Depress and hold Shift + mouse wheel, and drag.
